## 📱 使用方法

### 第一次使用（手机）

1. **打开APP**
   - 如果你已经部署到Netlify，用手机浏览器打开链接
   - 会自动跳转到登录页面

2. **登录**
   - 输入任意手机号/邮箱（演示模式，无需真实账号）
   - 输入任意密码
   - 点击"登录"按钮

3. **进入首页**
   - 登录成功后，自动跳转到任务列表首页
   - 首页左上角会显示你的用户名

4. **退出登录**
   - 点击底部导航的"我的"
   - 滑动到最下面，点击"退出登录"
   - 确认后退出，回到登录页

## 🔧 技术实现

### 登录状态检查
在 `index.html` 中：
```javascript
if (localStorage.getItem('isLoggedIn') !== 'true') {
    window.location.href = 'login.html';
}
```

### 保存登录状态
在 `login.html` 中：
```javascript
localStorage.setItem('isLoggedIn', 'true');
localStorage.setItem('username', username || '用户');
```

### 清除登录状态
在 `profile.html` 中：
```javascript
localStorage.removeItem('isLoggedIn');
localStorage.removeItem('username');
```
