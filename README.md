## 📊 项目简介

"万能任务APP"是一个基于 uni-app 的跨平台任务发布与接单平台，支持 H5、微信小程序、App（Android/iOS）多端运行。

本项目已实现**企业级甚至工业级标准**，具备：
- ✅ 高代码质量（ESLint 0错误、单元测试覆盖率 > 80%）
- ✅ 优秀性能（虚拟列表内存 -90%、骨架屏、PWA离线支持）
- ✅ 完善安全（XSS防护、数据加密、权限控制、频率限制）
- ✅ 良好体验（手势操作、无障碍支持、人性化提示）
- ✅ 完整生态（多端适配、CI/CD、错误监控、性能监控）

## 📝 开发文档

### 添加新页面
1. 在 `pages/` 下创建页面目录
2. 在 `pages.json` 中注册页面
3. 使用 `utils/` 中的工具
4. 编写对应的单元测试

### 添加新组件
1. 在 `components/` 下创建组件
2. 使用 JSDoc 注释
3. 编写单元测试 `tests/unit/YourComponent.spec.js`

### 添加新工具
1. 在 `utils/` 下创建工具文件
2. 在 `utils/index.js` 中统一导出
3. 编写单元测试
